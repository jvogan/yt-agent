# Support

## Where to Ask for Help

- Bug reports: open an issue with reproduction steps
- Feature requests: open an issue describing the user workflow you want to improve
- Security concerns: follow [SECURITY.md](SECURITY.md)

## Before Opening an Issue

- run `yt-agent doctor`
- confirm your `yt-dlp` install is current
- include the command you ran, your platform, and the relevant error output
